About Us

Pokenom

Pokenom is a tool for Pokemon go trainers to use when they want to find pokemon in their area, but are unsure where to start looking. Using this tool they can search for pokemon types around them and the map will show locations where they might spawn. All the user needs to do then is choose the location and head there.
